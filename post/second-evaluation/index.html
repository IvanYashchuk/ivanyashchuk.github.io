Second evaluation is passed!

Jul 30, 2017 00:00 · 97 words · 1 minute read

This week was the second evaluation week, which was passed successfully! During this month I’ve added new reference cells (UFCQuadrilateral and UFCHexahedron) to FIAT as well as the new wrapper class which defines TensorProductElement on new reference cells. Changes to DOLFIN and FFC were added so the definition of quadrilateral and hexahedron cells is the same across FEniCS. Solving Poisson equation works both for quadrilateral and hexahedral mesh now!

Plan for the future:

  • Find functionality that is not working with quadrilateral and hexahedral cells.
  • Add unit tests.
  • Add regression tests.
  • Documented demos.
  • Serial and parallel perfomance testing.
